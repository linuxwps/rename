# Requirements: Rename

**Defined:** 2026-05-13
**Core Value:** 让日常文件整理变得高效简单，一键批量重命名，错了还能撤销。

## v1 Requirements

Requirements for initial release. Each maps to roadmap phases.

### File Management

- [ ] **FILE-01**: 用户可以通过拖拽将文件从 Finder 拖入应用
- [ ] **FILE-02**: 用户可以通过文件选择器对话框选择文件
- [ ] **FILE-03**: 文件列表显示文件名、扩展名、文件大小、修改日期
- [ ] **FILE-04**: 用户可以从列表中移除单个文件
- [ ] **FILE-05**: 用户可以清空文件列表

### Preview

- [ ] **PREV-01**: 实时显示原文件名 → 新文件名的对比
- [ ] **PREV-02**: 高亮显示变化部分（新增绿色，删除红色）
- [ ] **PREV-03**: 检测并显示重名冲突

### Rename - Sequential (按时间顺序)

- [ ] **SEQ-01**: 用户可以按文件修改时间排序后重命名
- [ ] **SEQ-02**: 用户可以设置序号起始值
- [ ] **SEQ-03**: 用户可以设置序号位数（如 001 vs 1）
- [ ] **SEQ-04**: 用户可以选择序号位置（名前/名后）

### Rename - Regex (正则替换)

- [ ] **REGE-01**: 用户可以输入正则表达式进行匹配
- [ ] **REGE-02**: 用户可以输入替换文本（支持捕获组）
- [ ] **REGE-03**: 用户可以选择是否区分大小写

### Rename - Prefix (添加前缀)

- [ ] **PREF-01**: 用户可以输入前缀文本
- [ ] **PREF-02**: 前缀添加在原文件名前

### Rename - Suffix (添加后缀)

- [ ] **SUFF-01**: 用户可以输入后缀文本
- [ ] **SUFF-02**: 后缀添加在原文件名后、扩展名前

### Rename - Extension (修改拓展名)

- [ ] **EXT-01**: 用户可以修改文件扩展名
- [ ] **EXT-02**: 用户可以移除文件扩展名
- [ ] **EXT-03**: 用户可以添加文件扩展名

### Rename - Replace (替换文字)

- [ ] **REPL-01**: 用户可以输入查找文本
- [ ] **REPL-02**: 用户可以输入替换文本
- [ ] **REPL-03**: 用户可以选择是否区分大小写

### Execution

- [ ] **EXEC-01**: 用户点击执行按钮后原地覆盖重命名
- [ ] **EXEC-02**: 执行前验证所有文件名有效
- [ ] **EXEC-03**: 执行后更新文件列表

### Undo

- [ ] **UNDO-01**: 用户可以撤销最近一次批量重命名
- [ ] **UNDO-02**: 撤销恢复原文件名和位置

## v2 Requirements

Deferred to future release. Tracked but not in current roadmap.

### Metadata

- **META-01**: 支持使用照片 EXIF 日期重命名
- **META-02**: 支持使用照片相机型号重命名

### Advanced

- **CHAIN-01**: 支持多步骤链式重命名
- **PRESET-01**: 支持保存重命名预设
- **PRESET-02**: 支持加载重命名预设
- **CASE-01**: 支持大小写转换（全部大写/小写/首字母大写）

## Out of Scope

Explicitly excluded. Documented to prevent scope creep.

| Feature | Reason |
|---------|--------|
| 云同步功能 | 非核心，后期考虑 |
| 批处理脚本导入 | 复杂度高，v1 暂不包含 |
| AI 内容分析 | 需要外部依赖，非 pattern-based |
| 文件夹监控 | 后台处理增加复杂度 |
| 跨平台 (Windows) | 目标平台仅为 macOS |
